P5 - HTML5 presentation framework

P5

HTML5 presentation framework

Features

  • 100% HTML5, CSS and JavaScript (MooTools)
  • Presenter mode
  • Programmable slides
  • Built-in code highlighting (google-code-prettify)

How to use

  1. Open index.html and edit/add slides (<div class="slide">)
  2. Add styling to style.css and print.css
  3. If needed, programm the slides (see next slides)
  4. Run the presentation (index.html) in Firefox 3.6+, Safari 5+, Opera 9.60+ or Chrome 5+

Programming presentations

  • A show event is fired on the slides when they are shown and a hide event when hidden
  • A slidechange event is fired on the window object when slides change

Programming example

// Execute some complicated thing only while slide 7 is visible:
var timer;
$('slide7').addEvent('show', function(){
	timer = someComplicatedThing.periodical();
});
$('slide7').addEvent('hide', function(){
	$clear(timer);
});

Programming demo

Note that the slide events do not fire in the presenter view.

Presenter mode

Click the "Presenter view" link in the footer area to open the presenter view in a new window. The presenter view shows a timer and lets you see the currently displayed slide and which slide comes next.

Batteries (for geeks) included

P5 comes with google-code-prettify, which allows syntax highlighting of source code snippets in a HTML page. Just add class="prettyprint" to your <pre> and <code> elements.

Troubleshooting and bugs

  • If slides appear with the wrong font size in the presenter, refresh the presenter view
  • The presenter doesn't work in Chrome when accessed via file://
  • To prevent caching issues (the iframes in the presenter view might show an old, cached version of your presentation) clear your browser's cache before presenting.
